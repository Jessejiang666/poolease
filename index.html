01

市场与机会

泳池清洁机器人是典型「高客单、强场景、重信任」的品类。欧美有庭院泳池的家庭是核心人群，决策受 真实演示视频 与 红人测评 影响极大——这正是 POOLEASE 当前最该补的位置。

品类规模

一个稳定增长、且高度视觉化的赛道

2025 年全球泳池清洁机器人市场约 34.2 亿美元，预计 2034 年达 66.3 亿美元。增长不爆炸，但需求刚性、复购配件稳定，适合做长期品牌资产。

关键洞察：该品类「无线 / 无绳」已成主流卖点，用户最关心清洁力、越障、续航、app 体验——这些都必须靠 实拍与渲染 讲清楚，而 POOLEASE 当前最缺的正是素材。

03

品牌定位与策略拆解

一句话主张 → 支撑点 → 内容支柱，自上而下拆解，让每个人知道自己在建什么。

Brand Idea

主主张

“See the clean. Own the easy.”
看得见的干净，拥有即轻松

把「清洁效果可见」作为品牌记忆锚点——所有素材第一帧就要让人看到水变清。

支撑点 RTB

凭什么信

  • 无线无绳 · 一键启动
  • 实测清洁力（前后对比视频）
  • app 智能路径 / 续航
  • 独立站专属质保+配件
Content Pillars

内容三支柱

  • Proof 实测/前后对比
  • Life 庭院生活方式
  • How-to 使用/维护教程

SWOT 拆解

✅ 优势 S

  • 已在做红人建联，有先发关系
  • 独立站首页基本完成，地基在
  • 亚马逊有现成销售与评论背书
  • 团队小、决策快、易统一调性

⚠️ 劣势 W

  • 实拍/渲染素材严重短缺（最大短板）
  • 缺拍剪岗，素材产能靠外包/临时
  • 品牌视觉规范未系统沉淀
  • 独立站尚未跑通广告转化闭环

🌊 机会 O

  • 品类增长稳、视觉化强，适合内容打法
  • 竞品高价留出的质价比空档
  • 红人实测是低成本建信任的杠杆
  • 独立站可沉淀用户资产/复购

🔥 威胁 T

  • Aiper 等已占红人心智，需错位
  • 关税/物流/售后成本波动
  • 低价白牌在亚马逊价格战
  • 素材若粗制，反而伤品牌

05

素材基建（最大短板攻坚）

「现在主要是缺少实拍和渲染的图片、视频」——这是全案的第一杠杆。先有弹药，红人、社媒、广告才能开火。

实拍清单

泳池场景实拍（拍剪到岗后首批）

  • 15s 前/后对比（脏水→清澈，主视觉）
  • 产品下水/上岸 30s 旗舰片
  • 越障/爬壁/边角清洁特写 ×3
  • app 操控界面录屏
  • 庭院生活方式氛围片（家庭/宠物）
  • 开箱+安装 60s 教程
渲染清单

CG / 渲染资产

  • 产品白底图（全角度，独立站用）
  • 爆炸图/结构透视（讲技术）
  • 水下光效氛围图（社媒封面）
  • 尺寸/场景合成图（对比人居）
  • 动效短片（logo+水流过渡）
素材 SOP（拍剪到位即建）：① 建「素材中台」文件夹（原始/精修/社媒竖版/独立站横版/红人二剪包）；② 每条素材带「用途标签+尺寸规范」；③ 月度复盘素材使用率，淘汰低效类型。让一次拍摄产出多平台资产。

独立站成交路径（广告 × 社媒接力）

目标不是「在独立站卖」，而是「让红人/社媒/广告把人送到独立站并成交」。下面是把成交从亚马逊迁出的转化漏斗（方案期末月度目标态，示意）。

转化漏斗（月度目标）

* 示意目标，用于设定各层优化 KPI；实际以独立站看板为准。

三段接力模型

流量怎么一步步走进独立站

① 红人/社媒（建信任）
红人实测 + 官方内容种草，给独立站专属折扣码/链接，把「被说服的人」导过去。

② 付费广告（扩规模）
Meta/Google 承接搜索与兴趣流量，落地页与红人内容同源，降低跳失。

③ 再营销/邮件（收口）
加购未付、留资用户用邮件/SMS/再营销广告召回，拉升成交占比。

迁移节奏：初期独立站与亚马逊同价、独立站给「专属质保/配件/会员价」做差异，不让用户觉得独立站更贵，慢慢把成交权重移过去。

分阶段路线图（2026.07 → 2027.01）

不求快速，但求有变化。六个阶段层层递进，每个月都有可交付物。

P0 地基07.07–08底
补齐素材产能 + 视觉规范。 拍剪到岗、VI 定稿、首批实拍+渲染、独立站首页收尾与产品页上线。 ✓ 交付：素材库 v1 / VI 手册 / 产品页
P1 立信08–09中
内容矩阵 + 红人放量。 社媒内容日历跑起来、红人首批 50 寄样、UGC 机制、独立站 SEO 基础。 ✓ 交付：内容日历 / 红人首批内容 / 博客基础
P2 引流09–10中
广告冷启动 + 独立站承接。 Meta/Google 广告开投、红人专属链接引流独立站、落地页 A/B。 ✓ 交付：广告账户跑通 / 引流码机制
P3 转化10–11中
漏斗优化 + 召回。 邮件/SMS 序列、再营销、加购/结账优化、红人内容二剪分发。 ✓ 交付：自动化召回流 / 漏斗看板

复利。 复盘+素材资产归档、会员/社群冷启动、2027 品牌规划。 ✓ 交付：资产库 / 会员机制 / 明年 plan

预算分配（示意）

按「先补素材、再放量、后放大」的逻辑分配（占品牌+获客总盘比例，示意，便于排优先级）。

原则：前两个月预算向「素材+拍剪」倾斜（不然后续所有渠道都空转）；P2 起向红人与广告倾斜；P4 大促前预留促销/活动预算。
10

风险与里程碑

主要风险
  • 拍剪到岗延迟 → 用外包+供应商渲染过渡，品牌负责人审片兜底
  • 素材粗制伤品牌 → 设素材 SOP 与审片门槛，宁少勿糙
  • 广告烧钱没转化 → 先小预算验证落地页，再放量
  • 红人内容不同源 → 给红人「素材二剪包」，统一调性
关键里程碑
  • 07月底 拍剪到岗 + VI 定稿
  • 08月底 素材库 v1 + 产品页上线
  • 09月底 红人首批内容 + 广告冷启动
  • 11月 黑五/网一大促 + 占比爬坡
  • 01月 复盘 + 2027 规划
